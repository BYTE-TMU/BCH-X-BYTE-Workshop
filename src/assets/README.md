# src/assets/

Static assets imported directly by React components via ES module imports.

---

## Files

| File | Usage |
|---|---|
| `hero.png` | Scaffold placeholder from Vite — not used in the current site. Can be deleted. |
| `typescript.svg` | Scaffold placeholder from Vite — not used. Can be deleted. |
| `vite.svg` | Scaffold placeholder from Vite — not used. Can be deleted. |

---

## Notes

- Assets imported here are processed by Vite at build time (hashed filenames, optimised).
- Assets that should be served as-is (without hashing) belong in `public/` instead — e.g. `favicon.svg`.
- This folder is currently empty of active assets. Add workshop imagery here if needed in future.
